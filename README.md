## Quickstart
  ```sh
git clone https://github.com/koltis/prueba-tecnica.git
  ```

## Development

- Start the Mysql Database 

  ```sh
  npm run docker
  ```

- Initial setup:

  ```sh
  npm run build
  ```

- Run the  seed

  ```sh
  npm run seed
  ```

- Start dev server:

  ```sh
  npm run dev
  ```
